#### CSS Reset 
> 브라우저마다 기본 CSS값들이 입력되있어서 Reset을 해주어야함  
#### 사용법
><a href="https://www.jsdelivr.com/package/npm/reset-css">reset css 페이지</a>
>1)접속 후 reset.min.css html copy  
>2)head태그 안에 붙여넣기

#### 글자와 상자
>인라인요소 : 글자를 만들기 위한 요소    
>- 요소가 수평으로 쌓인다.  
>
>블록요소 : 상자(레이아웃)를 만들기 위한 요소
>- 요소가 수직으로 쌓인다.
